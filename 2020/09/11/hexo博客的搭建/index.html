第二步

  • 安装hexo

第三步

  • 建立一个文件夹存放所有的博客文件，错了就删了重来就行

初始化博客

看到如下就是初始化完成哒

可在本地端口打开

复制网址到在浏览器就能在本地预览了

一些基础命令

这是框架自动生成的博客就叫helloworld

hexo new “My New Post”——生成新的文章页

hexo s(server)——本地启动，简写s就行

hexo g(generate)——重新生成，基本就是修改之后hexo
clean 然后重新生成

hexo d(deploy)——部署到远端

第四步

  • 通过Github把博客部署到远端
    • 要用远端部署需要先下载一个git部署插件

如下：

  • 登录github建立新的仓库
    • 注意就仓库和ID要一致 .gihub.io
  • 然后就需要设置啦
    +找到如下文件，文件里找到最后面

type: git

rerp(就是远程仓库哒): bulabula……

branch(分支): master

ps:分支好像默认就是master，还有记得冒号后面跟空格

然后就可以hexo d往远端推了

关于换主题那些事情

  • 找到想要的主题下载下来就好啦 git clone
    • 之后找到配置文件更改主题就好啦

默认就是搭建的时候自动下载的主题哒

landscape删掉换成新主题就好啦

然后就能生成本地预览查看是否设置哒

然后hexo d就能推到远端啦
