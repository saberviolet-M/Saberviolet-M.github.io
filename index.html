Code State

2021-04-10

2xx

state code 原因短语 说明
200 OK 服务端正确接收并处理
201 Created post、put等添加请求实现
202 Accepted 服务器正确接收，但尚未完成，且不保证完成
204 No Content 成功了，但是响应报文没有主体部分
206 Partial Content 范围请求成功（分段下载，断点续传）

3xx

state code 原因短语 说明
301 Moved Permanently 永久重定向（http->https）
302 Found 临时重定向(登录操作->登录页->登录完了跳回来)
303 See Other post被请求成功过了，用get请求
304 Not Modified 认证资源有效（协商缓存），不返回数据
307 Temporary Redirect post没处理，正常post请求

4xx

state code 原因短语 说明
400 Bad Request 请求报文存在语法错误
401 Unauthorized 没有权限
403 Forbidden 请求资源被服务器拒绝
404 Not Found 服务器上没有资源
408 Request Timeout 客户端超时
409 Confict 请求资源冲突

5xx

state code 原因短语 说明
500 Internal Server Error 服务器执行错误
501 Not Implemented 超出服务器范围，不支持某个方法
502 Bad Gateway 代理服务器请求远端服务器无响应
503 Service Unavailable 服务器超载，停机维护
504 Gateway Time-out 代理服务器请求远端服务器超时
505 Http Version Not Supported 服务器不支持，或拒绝支持请求的http版本

动词

2021-03-31

构造

  • 词干 + 语态（主动、被动） + 体（动作完成、动作进行、动作结果存留） + 肯/否 + 时态
  • 読ま + れ + てい + なかっ + た （还没有被读过）

自动词/他动词

  • 自动词 – 不需要宾语（不及物动词）[动作、状态]
  • 他动词 – 需要宾语（及物动词）[行为]

动词自、他动词中的词性偏向

他动词 >>[偏向]>> [自动]

[他动词]す >> eる >> aる[自动]

[他动词]eる >> 五段动词[他动词]

[他动词]壊す >> 壊れる[自动]

[他动词]始める >> 始まる[自动]

[他动词]つける >> つく[自动]

但有很多例外

  • ~が[自动]
  • ~を[他动词]

动词活用（五段动词）

  • 未然形 – あ （あ　>>>　わ）
  • 连用形 – い （**+**用言、助动词、小句）
  • 终止形连体形 – う （*+**名、代 == 体言）
  • 假定形*命令形 – え
  • 未然形 – お

动词活用（一段动词）【汉字词+る是五段动词】

  • 未然形 – い、え
  • 连用形 – い、え
  • 终止形*连体形 – い、え + る
  • 假定形*命令形 – い、え + れ
  • 未然形 – い、え + ろ/よ

动词活用（カ变、サ变动词）

  • 未然形 – 来(こ) [カ变] || し、せ、さ[サ变]
  • 连用形 – 来(き) [カ变] || し[サ变]
  • 终止形*连体形 – 来(く)る || [カ变] する[サ变]
  • 假定形*命令形 – 来(く)れ || [カ变] すれ[サ变]
  • 未然形 – 来(こ)い [カ变] || しろ/せよ[サ变]

grid栅格布局

2021-03-26

开启grid容器

  • display: grid
  • display: inline-grid

容器划分

  • grid-template-rows(columns): [数字]px 100px 200px 50px
  • grid-template-rows(columns): [百分比]% 33% 33% 1%
  • grid-template-rows(columns): [百分比]% 33% 33% 1%
  • grid-template-rows(columns): repeat([重复的次数], 重复的值)
  • grid-template-rows(columns): repeat(2, 100px 200px 50px)

名词谓语句

2021-03-26

名词不能直接做谓语

  • ~だ – 是 – 常体
    • 只接句号或终助词
    • 否定要改成で
  • ~ではない – 不是
  • （か？）– 疑问 – 终助词
  • ~です　– 助动词 – 敬体
  • ~ではありません
  • ~ ですか？
  • ~ である – 演讲等正式场合 – 无否定
  • ~ であります – 无否定
  • ~ でございます – 敬语 – 无否定
  • ~ は/が – 副助词は/格助词が
  • は – 主题 – 强调后为新信息
  • が – 强调前为新信息
  • も – 副助词，判断、也
